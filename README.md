## Installation

ParallaxScrollView is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
# Podfile
use_frameworks!

target 'YOUR_TARGET_NAME' do
  pod 'ParallaxScrollView', '1.0.3'
end
```
